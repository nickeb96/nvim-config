
How to Update Spell Files
-------------------------

1.  `cd` to `~/.config/nvim/spell/` first
2.  open neovim
3.  type `:mkspell! personal`
    + `personal` is the name of the locale
    + The `!` overwrites the existing `locale.spl` file
